# How to Run

1. **Create a local MySQL database.**

2. **Create a `.env` file with the following contents:**

```bash
DB_HOST=localhost
DB_USER=root
DB_PASSWORD="{your password}"
DB_NAME="{your db name}"
OPENAI_API_KEY="{your openai api key}"
```

3. **Start the backend:**

```bash
npm start
```

4. **Start the frontend:**

```bash
npm start
```
